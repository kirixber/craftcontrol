// What it does

Everything your SMP needs

No more pinning IPs in chat. No more forgetting coords. CraftControl keeps your whole server organized.

📡
Live Server Status
Real-time ping, player count, version, and network stats including packet loss and jitter.
*status*ping*online
📍
Shared Coords
Save and share base locations, portals, and waypoints across Overworld, Nether, and End.
*coords add*coords list
🔨
Crafting Recipes
1,100+ Minecraft recipes with wiki descriptions, usage info, and smart typo suggestions.
*recipe*wiki
🎵
MC Music in VC
Play any Minecraft sound or music disc in your voice channel. Loop, browse 100+ sounds.
*play pigstep*loop
⚡
RCON Commands
Run in-game commands from Discord. Gamemode, kick, ban, whitelist, teleport, and more.
*gm survival*kick*wl add
🌐
Multi-Server
Manage multiple Minecraft servers from one Discord. Each gets its own data and coords.
*status smp1*ip smp2
🎨
Pixelate Images
Convert any image into a Minecraft-style pixelated version. Attach or paste a URL.
*pixelate
🧩
Mod List
Maintain a list of required and optional client mods with download links for your players.
*mods list*mods add
👤
Player Lookup
Look up any player's skin, UUID, and online status. Works for premium and cracked accounts.
*player Steve

// Full command list

Every command, at a glance

Prefix: * — All commands support multiple server targeting.

🌐 Server Info
  • *ipShow connection details
  • *statusLive status + player count
  • *onlineWho's currently playing
  • *pingNetwork stats, packet loss, jitter
📍 Coordinates
  • *coords listView all saved waypoints
  • *coords addSave a location
  • *coords deleteRemove a waypoint
📚 Minecraft Info
  • *recipeCrafting recipe + wiki info
  • *wikiSearch Minecraft Wiki
  • *playerPlayer skin, UUID, status
  • *modsClient mod list
🎵 Audio
  • *playPlay MC sound in VC
  • *loopLoop a sound
  • *soundsBrowse all sounds
  • *stopDisconnect from VC
🛡️ Moderation (RCON)
  • *whitelistManage whitelist
  • *ban / *unbanBan management
  • *kickKick a player
🎮 Server Commands (RCON)
  • *gmChange gamemode
  • *gameruleGet/set gamerules
  • *tpTeleport players
  • *giveGive items
  • *time / *weatherWorld settings
  • *msgDM online players in-game
// Get started

Up in 3 steps

01

Add CraftControl to your server

Click the invite button and select your Discord server. The bot joins instantly.

02

Run *setup in any channel

The bot walks you through adding your Minecraft server — IP, port, version. No config files needed.

03

Start using commands

Try *status, *ip, *recipe diamond sword. For in-game commands run *rconguide.
